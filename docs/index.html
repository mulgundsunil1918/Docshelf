What makes DocShelf different

Other document apps treat your files like cloud assets. DocShelf treats them like documents — yours, on your device, organised your way, with the features you'd actually use.

📷

On-device document scanner

Tap Scan, point at any paper, the camera auto-detects edges and corrects perspective — like Adobe Scan but fully offline. Multi-page captures stitch into a single PDF automatically. Powered by Google ML Kit on-device.

📥

Import from anywhere

Pick a single file, multiple files, or an entire folder. Or share from any other app — WhatsApp, Drive, Gmail, Files — straight into DocShelf. The "Find on device" scanner crawls common folders for files you already own.

🗂️

14 starter folders, fully customisable

Identity, Finance, Work, Education, Health, Insurance & Policies, Property, Vehicle, Bills, Receipts & Warranties, Quotations & Estimates, Travel, Family, Other. Add subfolders. Add categories. Delete what you don't need.

✏️

Rich-text notes built in

Bold, italic, underline, strikethrough, highlight, bullet & numbered lists, headings, indent, quote, undo / redo. Pick a sticky-note background colour. Notes save alongside your documents in the same folder tree.

⏰

Calendar reminders for what expires

Toggle "expiry date" on a passport, insurance, lease, license, NDA, software subscription. DocShelf hands the reminder to your phone calendar; the OS delivers it. Survives reboots, battery savers, and OEM kill behaviour.

🔍

Search that actually finds

Search across file names, your descriptions, AND the folder paths each file lives in. Filter by file type — PDFs / images / notes / videos / docs. Find any document in seconds.

🌙

Material 3 with proper dark mode

Indigo-and-amber Material 3 design, Nunito throughout. Light + dark modes both designed intentionally — not a slap-on tint. Every screen reads cleanly at midnight or noon.

🌐

Local-first, forever

No accounts. No upload. No sync. No tracking. No ads. Files live in /storage/emulated/0/DocShelf/ on your device — visible to your file manager, browseable like any other folder. Files survive uninstall.

How it works

  1. Install & tap through a 4-slide tutorial. No account, no signup, no PIN — your phone's own lock screen is the only meaningful security boundary anyway.
  2. Import documents. File picker (single / multiple / folder), share from another app, scan paper with the camera, or run "Find on device" to pull in files you already have.
  3. Tag with category + optional expiry. 14 starter folders cover most things; add your own subfolders. If a doc expires, toggle the date — the calendar event opens pre-filled, you confirm once.
  4. Search, bookmark, share, repeat. Bookmarks surface on Home. Search runs across names, descriptions, and folder paths. Long-press any file to share via the standard Android sheet.
  5. Stay private. No cloud. No telemetry. Every file stays in /DocShelf/ on your device. No account to create, nothing to sign in to, nothing to forget.

Privacy you can actually verify

DocShelf does not have a server. There is no telemetry, no analytics, no cloud sync, no in-app purchases, no ads, no accounts. The only network calls the app makes are when you tap an outbound link (Privacy Policy, Rate the app, mailto).
