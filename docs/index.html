What makes DocShelf different

Other document apps treat your files like cloud assets. DocShelf treats them like documents — yours, on your device, organized your way.

🪐

Spaces — context switching

A Space is a top-level context: yourself, a family member, work, a side project, a class you teach. Each Space has its own folder tree on disk. Switch with one tap; all data filters instantly.

🗂️

14 starter categories, fully customizable

Identity, Finance, Work, Education, Health, Insurance, Property, Vehicle, Bills, Receipts & Warranties, Quotations, Travel, Family, Other. Add subfolders. Add categories. Delete what you don't need.

⏰

Calendar-based expiry reminders

Toggle "expiry date" on any file — passport, insurance, license, NDA, lease, software license. DocShelf hands the reminder to your phone calendar; the OS delivers it. Survives reboots and battery savers.

📥

Import from anywhere

Long-press a WhatsApp / Gmail / Drive PDF → Share → DocShelf. Or batch-import a whole folder. Or scan device folders (WhatsApp, Telegram, Downloads, Documents) for files you already have.

🔍

Search that actually finds

Search across file names, your descriptions, AND the folder paths they live in. Filter by file type. Filter by Space. Find that one PDF in seconds.

🌐

Local-first, forever

No accounts. No upload. No sync. No tracking. No ads. Files live in /storage/emulated/0/DocShelf/ on your device — back them up however you like.

How it works

  1. Install + set up your Spaces. First launch: add yourself, then any family / work / study / project Spaces you want.
  2. Import documents. File picker, share-from-anywhere, or scan WhatsApp / Drive / Downloads. Tag with a category + Space.
  3. Add expiry dates where it helps. DocShelf adds the event to your phone calendar — your normal reminder workflow takes over.
  4. Bookmark, search, switch. Frequently used docs on Home. Search across everything. Switch Spaces with one tap.
  5. Stay private. No account. No cloud. Files in /DocShelf/ on your device.

Privacy you can actually verify

DocShelf does not have a server. There is no telemetry, no analytics, no cloud sync, no in-app purchases, no ads, no accounts. The full source is at github.com/mulgundsunil1918/Docshelf — read it, audit it, fork it.
