Drop a 1024x1024 PNG named `app_icon.png` in this folder, then run:

    flutter pub run flutter_launcher_icons

That regenerates all Android launcher icons (mipmap-mdpi → xxxhdpi).

Until you add a real icon, the build uses the default Flutter blue icon.

Recommended: indigo background (#3D5AFE) with a white folder + amber
checkmark/shield to match the in-app brand.
