Why DocShelf is different

Other document apps treat your files like cloud assets. DocShelf treats them like documents — the way Indian middle-class households actually need them organized.

🪪

Categories that match real life

Identity (Aadhaar, PAN, Passport). Finance (ITR, Bank, MF). Vehicle (RC, PUC). Property (sale deed, will). 11 root folders, all the subfolders you actually have.

👨‍👩‍👧

Family profiles

Self, spouse, kids, parents — separate document trees per person. Tap an avatar at the top to switch instantly. Every file knows whose it is.

⏰

Expiry reminders

Set an expiry date on your passport, license, PUC, insurance — DocShelf notifies you 30 days before. Configurable per-document. Never miss a renewal again.

🔐

Bank-grade lock

Biometric (fingerprint / face) plus a 4-digit PIN backup. Auto re-locks after backgrounding. 5 wrong PIN attempts triggers a 30-second cooldown.

📥

Import from anywhere

Long-press a WhatsApp PDF → Share → DocShelf. Or scan WhatsApp, Drive, Telegram, Downloads for files you already have. Or batch-import a whole folder.

🌐

Local-first, forever

No accounts. No upload. No sync. No tracking. Files live in /storage/emulated/0/DocShelf/ on your device — back them up however you like.

How it works

  1. Install + add family. First launch: pick avatars for yourself, spouse, kids, parents.
  2. Set a PIN. Backup for biometric. Encrypted with SHA-256, never leaves the device.
  3. Import documents. File picker, share-from-anywhere, or scan WhatsApp / Drive / Downloads.
  4. Tag with category + expiry. 30-day reminder by default. Bookmark frequently used ones.
  5. Find anything. Search across names, descriptions, folders. Filter by type. Switch members instantly.

Privacy you can actually verify

DocShelf does not have a server. There is no telemetry, no analytics, no cloud sync, no in-app purchases, no ads. The full source is at github.com/mulgundsunil1918/Docshelf — read it, audit it, fork it.
